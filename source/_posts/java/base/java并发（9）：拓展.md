# 死锁

死锁的必要条件

- 互斥条件
- 请求和保持条件
- 不剥夺条件
- 环路等待条件

解决锁

- 持有锁一定时间，依然在等待的话，则释放持有的所有锁。

# 多线程并发最佳实践

- 尽量使用本地变量
- 使用不可变类
- 最小化锁的作用域范围
- 使用线程池的executor，而不是newThread
- 宁可使用同步也不要使用现场的wait和notify
- 使用blockingQueue实现生产-消费模式
- 使用并发集合而不是加了锁的同步集合
- 使用semaphore创建有界的访问
- 宁可使用同步代码块，也不使用同步的方法
- 避免使用静态变量
