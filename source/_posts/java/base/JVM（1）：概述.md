# Java技术体系

包括

- Java程序设计语言
- 各种硬件平台上的Java虚拟机
- Class文件格式
- Java API类库
- 来自商业机构和开源社区的第三方Java类库

JDK：

- Java程序设计语言
- Java虚拟机
- Java API类库
- 支持Java程序开发的最小环境

JRE

- Java SE API子集
- Java虚拟机
- 支持Java程序运行的标准环境
