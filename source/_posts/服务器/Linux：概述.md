# 开源软件

开源软件，即获得程序的源代码，而不是编译后的二进制文件

- 使用的自由：绝大多数开源软件免费
- 研究的自由：可以获得软件源代码
- 散步及改良的自由：可以自由传播、改良甚至销售

在Linux当中，以服务器端的角度看，Linux当中有更多的开源软件，有着更高质量的软件

- Apache：网站服务搭建软件
- Nginx：占用服务器资源更少，支持更高的并发
- MySQL
- php
- Samba
- mongoDB
- python
- Ruby
- Sphinx等
